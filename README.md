# ExpenseFlow - App for expense tracker

## Description
 ExpenseFlow app works as a tracker for the user who is interested in tracking down their money spend every days, weeks or months and years.
 
 ## Features
 1. Simple and clean interface.
 
 2. Add and update the expenses.
 
 3. Add and update with date, category and summary.
 
 4. Filter the list with category
 
 5. Change the display mode with light and dark.
 
 6. Delete individual expense or clear all data expenses.
